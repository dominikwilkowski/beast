## How to install

Download the game from this repo and install it's dependencies via:

```shell
npm i
```

or

```
yarn
```

Then run:

```shell
node prod/index.js
```


## How to play

The goal is to squash all beasts in each level. To squash a beast between two or more blocks. Press `q` to quit.
